gridView.commit()
편집상태를 종료하고 데이터프로바이더와 동기화

gridView.getItemCount()
그리드뷰의 행 갯수 파악

getValue(index,'column')
그리드뷰,프로바이더 둘다 사용가능한 함수 / 특정 셀 값 가져오기

setValue(index, 'column', 'value')
그리드뷰,프로바이더 둘다 사용가능한 함수 / 특정 셀 값 입력

gridView.setCurrent({ dataRow: index, column: 'column' })
그리드뷰의 특정 셀 포커스
인덱스만 입력시 행만 선택

dataProvider.insertRows(0, [객체,])
여러행 입력가능. 보통 한행추가할때도 씀

dataProvider.getRowCount()
데이터프로바이더의 로우갯수 전부 파악(create,update,delete,c&d, none 가리지 않고 전부 파악)

dataProvider.getRowState(i)
데이터프로바이더의 해당 행 상태 확인 (create,update,delete,c&d, none)

dataProvider.getRowStateCount('created')
데이터프로바이더의 해당 상태의 행 갯수 파악

getJsonRows()
그리드뷰,프로바이더 둘다 사용가능한 함수
그리드뷰는 보여지는곳만, 프로바이더는 delete등 숨겨진 행까지 저장하고있는 데이터
행을 객체형식으로 저장하고 여러행을 배열로 저장해서 결국에 객체배열을 반환하는 함수
