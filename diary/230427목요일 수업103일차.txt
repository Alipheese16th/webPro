오늘 배운 꿀팁
dto에서 멤버변수값이 초기화될때 int 형은 기본적으로 숫자 0으로 초기화되기때문에
db에 null로 집어넣고 싶어도 if문으로 비교해가면서 해야되기때문에 귀찮다.

근데 Integer 형으로 멤버변수타입을 지정하면 값이 초기화될때 null로 초기화되기때문에 편하게 집어넣을수있다
단 마이바티스에서 그냥 null로 db에 집어넣으려고 하면 에러나니까 jdbcType=NUMERIC을 사용해야한다

근데 이것도 만능은 아닌게 select 문에서 where절로 검색할때 만약 null도 허용해서 검색하고싶다면 = null 로 검색할수가없다 IS NULL 로 검색해야됨
결국에 where절에 null 들어갈거같으면 그냥 if문 사용해야함
